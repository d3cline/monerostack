Welcome to the Monero MCP Server

Thank you for landing here. This server provides a Model Context Protocol (MCP) interface for Monero tools.

The MCP endpoint is available at: /mcp

The Github Repo is available at: https://github.com/d3cline/monerostack

MCP configuration (mcp.json)

Add this to your MCP host configuration and adjust the URL/port if needed.

{
  "servers": {
		"monerostack": {
			"url": "https://monerostack.d3cline.net/mcp",
			"type": "http"
		}
  }
